概要: `src/main.rs` に四則演算のサンプルコードを追加しました。変数名は要求どおりcamelCaseにしています。

追加したコード:

    // --- 四則演算のサンプル（変数はcamelCase） ---
    let firstNumber: f64 = 10.0;
    let secondNumber: f64 = 3.0;

    let sum = firstNumber + secondNumber; // 足し算
    let difference = firstNumber - secondNumber; // 引き算
    let product = firstNumber * secondNumber; // 掛け算
    let quotient = firstNumber / secondNumber; // 割り算

    println!("Arithmetic samples:");
    println!("{} + {} = {}", firstNumber, secondNumber, sum);
    println!("{} - {} = {}", firstNumber, secondNumber, difference);
    println!("{} * {} = {}", firstNumber, secondNumber, product);
    println!("{} / {} = {}", firstNumber, secondNumber, quotient);
